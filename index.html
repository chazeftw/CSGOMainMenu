F.A.Q

Will I get banned for this?

No.

What happens on game updates?

The file will be reset and if other weapons or items are added, it might not be the lines stated that is those to be replaced. Therefore search after the keyword given.

Should I backup the txt file before editing?

That's a good idea but if everything goes wrong you can always verify integrity of game cache:
Library > CS:GO > Right Click > Properties > Local Files > Verify dat shiet
